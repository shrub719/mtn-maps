# metronome.maps

Map files for [metronome](https://github.com/shrub719/metronome).

This is a repository of `.mtn` files, either written by hand or converted from
`.osu` files using [polyrhythm](https://github.com/shrub719/mtn-poly). They can be compiled to 
external map files (`.mtb`) with polyrhythm as well.

These are intended for use in the building of the base version of metronome as included binaries.
For more information, see [here](https://github.com/shrub719/metronome?tab=readme-ov-file#building).
